Tag vs Element vs Attribute

Tag

Which comes in < angular brakets >

we have two types of tag

  1. Which comes with start tag (<start >) and closing tag (</end >), for eg. h1, p, title etc
  2. Self closing tag , for eg. br, hr, meta etc

Element

Start tag - some content - end tag

Attribute

It provides some additional information by which behaviour of element changes. It always written in start tag
